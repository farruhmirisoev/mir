Основные файлы стилей расположены в папке scss. Ниже приведено описание файлов по папкам. Если в списке нет файла -
значит описание внутри файла

В случае проблем с поиском нужных стилей - воспользуйтесь поиском по директории scss

Папка components:
- buttons - основные стили кнопок сайта, разные варианты кнопок.
- carousel - основные стили слайдеров
- footer - собраны все элементы подвала
- grid - разметка по разным устройствам
- header - собраны все элементы шапки
- map - оформление карты (кнопки, поиск и прочие элементы)
- menu - стили меню в левом сайдбаре (врачи, акции, статьи)
- steps - стили минипанели "Как начать жить без боли и забот", стандартные шаги и пользовательские шаги (вьюшки)
- taxonomy - стили страниц таксономии
- titles - стили разных типов заголовков
- views - основные стили всех вьюшек.
- webforms - основные стили всех вебформ и стилизация некоторых уникальных вьюшек

Папка hover-effects:
- разные эффекты для элементов, которые можно включить при помощи @include название_эффекта

Папка includes:
- color - основные цвет проекта

Папка node_types:
- стили типов материалов и главных страниц с этими типами,
название файла - машинное название типа материала

Папка other:
- 404 - оформление страницы 404, not found
- modal - стили всплывающего окна
- tabs - стили для табов

main.scss - включает в себя остальные стили:
- подключение всех файлов стилей,
- главная страница,
- дополнительные классы,
- табы,
- хлебные крошки,
- страница контактов,
- всплывающее окно,
- таймер
